Upload a CSV or Excel file containing securities information.

File Format Requirements

Please ensure your file follows these guidelines:

  • security_id - String (max 13 characters) (required)
  • ISIN - Must start with 2-letter country code followed by 10 digits (required)
  • currency - 3-character ISO code (defaults to USD if not specified)
  • maturity_date - Date in YYYY-MM-DD format
  • issuer - Must match an issuer name already uploaded (required)
  • description - String (max 256 characters) (required)
Accepted formats: CSV, Excel (.xlsx, .xls)
